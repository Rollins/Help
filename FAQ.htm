How Are Phone and Web Leads Tracked?

Marketing Technology uses a conversion workflow to combine data from several sources. This workflow gathers records such as caller data, click data, campaign code, and revenue data, then matches and combines this data for viewing in the Marin platform.

What Should I Do if a Website is Down?

If a franchisee, consumer, or Rollins employee is unable to access International Franchise website, try the troubleshooting steps in International Franchise Site is Inaccessible, then escalate to the Marketing Technology team.

If you suspect that an Orkin website is down, escalate this to the Marketing Technology team by emailing MarketingTechAlerts@Rollins.com. See Orkin.com or Orkincanada.ca is Down for more information.

How Can I Ensure I Have the Latest Version of the Code Base?

To ensure you are always working on the latest code base, pull from the dev.orkin.com repository, resolve any conflicts, then push the code back to dev.orkin.com.

See Mitigating Risk in GitHub Version Control for more information.

Can I See the Details of Recent Commits in a GitHub Repository?

GitHub provides two methods of viewing the history of commit actions. See Viewing Commit History for more information.

My Website Isn't Loading Images. What Should I Do?

If a franchisee, consumer, or Rollins employee experiences image loading problems on a website, or recent updates are not displayed, follow the troubleshooting steps in Site Does Not Display Images or Updated Content
