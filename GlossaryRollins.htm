A

Activity Log

See Marin Activity Log.

adid (query string parameter)

A parameter that can be passed on the query string of the a URL to a Landing Page or other tracking-enabled site that specifies the marketing campaign ID to use. This parameter should only be used as a convenient way to test campaign activation and not used in any public URLs.

Adobe

Adobe (NASDAQ: ADBE, adobe.com) is an American computer software company headquartered in San Jose, California, USA. The company has historically focused upon the creation of multimedia and creativity software products, with a more-recent foray towards rich Internet application software development.

Adobe Reader

Adobe Reader is used to view PDF Files.

Advertising Medium

The means through which an advertisement is placed. For example, television.

Advertising Partner

A partner through which one or more ads are placed.

AdWords

See Google AdWords.

After Call Work (ACW)

At RCSC, after call work is work performed by an agent on an Edge case after a sales call has been completed but before the case is saved and forwarded for quality assurance. Agents are incentivized to enter complete and accurate details into each case before it is seen by the field.

Anchor Text

The clickable text of a hyperlink; in HTML code anchor text appears between the <a> and </a> tags. Anchor text is an important factor in Search Engine Optimization.

Associate Certified Entomologist (ACE)

A certification from the Entomological Society of America. A holder of an ACE certification presumably knows the basics of  entomology, pest identification, IPM principles, and more.

AutoClean

A bathroom care system from Orkin Commercial that chemically helps reduce bacteria build-up, prevent stains, scaling and odors, and reduce janitorial expenses. 

AutoFlush

A bathroom care system from Orkin Commercial that uses fail-safe infrared user-detection device that automatically flushes toilets after use, reducing sewer gas buildup and helping to prevent an offensive and unsanitary bathroom environment. The flushing system lowers the risk of cross-infection spread by touching fixtures and reduces damage to fixtures due to improper use. 

AutoFresh

A bathroom care system from Orkin Commercial that provides maximum odor control by dispensing precise neutralizing fragrances to help create a pleasant bathroom environment. 

Automatic Number Identification (ANI)

A feature of telephony intelligent network services that permits subscribers to display or capture the billing telephone number of a calling party. ANI has two components, information digits, which identify the class of service, and the calling party billing telephone number. ANI is unrelated to caller ID and a caller's telephone number and line type are captured by ANI service even if caller ID blocking is activated.

AutoPay

A system that allows customers to pay for each service automatically from their credit card or bank check card. This functionality was made available in Edge on September 29th 2008. 

Average Handle Time (AHT)

At RCSC, the average time it takes to handle a call plus the average after call work (ACW) time.

Azure

See Windows Azure.

B

Brand Affinity

The authority a brand holds with customers, how well customers identify with a brand's promise, and peer-group approval associated with using a brand.

Brand Architecture

Brand architecture is the structure of brands within an organizational entity. It is the way in which the brands within a company’s portfolio are related to, and differentiated from, one another. The architecture should define the different leagues of branding within the organization; how the corporate brand and sub-brands relate to and support each other; and how the sub-brands reflect or reinforce the core purpose of the corporate brand to which they belong.

Breadcrumb Trail

Sometimes simply referred to as breadcrumbs, a navigational aid used in user interfaces. It allows users to keep track of their locations within programs, documents, or websites. The term comes from the trail of breadcrumbs left by Hansel and Gretel in the popular fairytale.

C 

Call Management System (CMS)

The Rollins system through which telephone calls are routed, tracked, recorded, and otherwise managed.

Campaign

See Marketing Campaign, Google AdWords Campaign, and Bing AdCenter Ad.

Campaign Code

See Marketing Campaign ID.

Canonical URL

A reference to the preferred version of content for a set of pages with highly similar content. Canonical URLs are specified in HTML by adding a <link> element with the attribute rel="canonical" to the <head> section all pages.

Cascading Style Sheets (CSS)

A simple mechanism for adding style (e.g. fonts, colors, spacing) to web documents.

Case

See Edge case.

Click data

The information that may be associated with a "click" (or "tap" for touch screens), such as a timestamp of when the event occured. In Digital Marketing, this data is most often associated with referrer. SEM-driven clicks will often include query (search term) data, and PPC clicks will include bid keyword, creative, and match type information.

Cloud computing 

A style of computing in which dynamically scalable and often virtualized resources are provided as a service over the Internet. The term cloud is used as a metaphor for the Internet, based on how the Internet is depicted in computer network diagrams and is an abstraction for the complex infrastructure it conceals.

cm_mmc (query string parameter)

A parameter that can be passed on the query string of a URL that specifies how Coremetrics should track visits to the page. The cm_mmc parameter contains four sub-fields, delimited by "-_-". This cm_mmc parameter must include a marketing campaign ID for the page to render the correct phone number and other details shown to the user. The campaign ID is identified within the cm_mmc parameter as the only parameter that is ALL CAPS. When creating a new URL, be careful to ensure that the campaign ID is the only part of the cm_mmc parameter that is all caps. Additionally, spaces - though sometimes valid - should generally be avoided.

Cognos (http://www-01.ibm.com/software/analytics/cognos/)

A business intelligence, reporting, and analysis platform from IBM.

Concentrate (https://www.concentrateme.com/)

A tool from Juice Analytics that connects directly to Google Analytics to:

    • Visualize keyword relationships to see connections and performance of common word combinations.
    • Summarize performance metrics to compare how visitors behave based on individual words or filtered sets of keywords.
    • Consolidate queries that use navigational keywords and common entities (cities, states, etc.).

Content Management System (CMS)

A Content Management System, also known as CMS, is a system used to manage the content and features of a website. The CMS used by Marketing IT is called N2.

Cookie

A small string of text stored by a web browser containing state information for the browser to communicate to the website it originated from with each request.

Creative Program

The Creative Program is the field marketing program designed to drive leads through field technician interaction with prospective customers.

Customer Relationship Management (CRM)

Generally refers to the software and processes used to track and organize contacts with current and prospective customers. Rollins's CRM goals are to improve services provided to customers, and to use customer contact information for targeted marketing.

Customer Service Representative (CSR)

Company representatives that interact directly with the customer. Sometimes shortened to Customer Service Rep, Customer Rep, or CSR.

Data Mart

A dataset that is pre-aggregated for reporting purposes, typically for a specific line of business. Data for each data mart is gathered from ODS. Reporting platforms, such as Cognos, are used to report-on and analyze data from one or more data marts.

Any TV advertising that asks consumers to respond directly to the company; usually either by calling an 800 number or by visiting a web site.

The Domain Name System, commonly referred to as DNS, is a hierarchical naming system for computers, services, or any resource connected to the Internet or a private network (such as a website or intranet site). For a more complete description of how DNS works, view the DNS for non-technical people article on the Marketing IT wiki.

DoubleClick

Owned by Google, DoubleClick is a provider of digital marketing technology and services. The world's top marketers, publishers and agencies utilize DoubleClick's expertise in ad serving, rich media, video, search and affiliate marketing to help them make the most of the digital medium. From its position at the nerve center of digital marketing, DoubleClick provides superior insights and insider knowledge to its customers.

DRY Principle

The DRY (Don't Repeat Yourself) principle is a coding and design philosophy aimed at reducing repetition. The philosophy emphasizes that information should not be duplicated, because duplication increases the difficulty of change, often decreases clarity, and leads to inconsistency. Adherence to the DRY Principle is a coding standard of Marketing IT.

E

Edge

Edge is the RCSC database to which leads are submitted via WebMethods. Edge includes functions for basic sales and customer service, a "how’s my driving" interface with fleet system to identify vehicles and branches, a Termite Claims Handling interface with risk management systems to assign adjuster and track claims, a Termite Payments manager to record call-in payments, a Field Cross marketing leads program interface with payroll records to identify tech and track and pay for leads, Community Relations Fulfillment tracking of brochure requests etc to customers, a Case Data QA Process which selectively allows all new created cases to be sent to a holding queue to be checked for accuracy before being sent to the field, a Training Group Queue that allows training of agents on creation of cases without sending them live through the system, a surveys builder that allows building of survey templates within Edge and loading of call lists to progressively call customers to take surveys and track results, a Web Program interface with web methods and the data-tel dialer to process leads through the system from the web and present lead case details to agents after a connection is made with a prospect, an Auto Reporting module to automatically run selective reports (system and operations performance) at specific times and send them by email to selected individuals, and a module for the Lost Leads program that interfaces with data-tel dialer to load lists and present lead case details to agents after a connection is made with a prospect.

Edge case

A record of a customer contact in Edge, typically for sales or customer service inquiries.

Electronic Funds Transfer (EFT)

The exchange of money from one account to another through a computer system. Orkin uses Element for EFT payments.

Enterprise Data Warehouse (EDW)

A large-scale database used for reporting, modeling, and data analysis. 

Electronic Data for Sales (EDS)

The sales force management application used by Orkin.

Entry Page

Similar to a landing page, an entry page is the page at which a visitor enters a web site. Unlike a landing page, an entry page is not necessarily the result of clicking an ad. The terms entry page and landing page are often used interchangeably however.

F

Fact Table

In data warehousing, a fact table is a database table that contains the measurements, metrics, or "facts" associated with a business process.

FireFox

First Call Resolution (FCR)

The ability to solve a customer's issue on the first call.

FOCUS

Form 19

An overview, printed from FOCUS, of all of the customers due for service in a given month.  Basically a cheat sheet for the tech that shows them who they service that month and what day each customer is due.

Frequency

Frequency is the number of times your advertising schedule was exposed to an average person or household during a time period. For example, if your commercials, due to repetition, reached a total of 30 million households in a region that has 10 million households, the number would be 3.

FTE

A common shorthand for a full time employee.

A common shorthand used in Internet and SMS communications meaning "for what it's worth".

Gross Rating Point (GRP)

A measurement of audience size (viewership or listenership) to quantify a medium's ability to reach a target audience. GRPs estimate the number of people a communication will reach, irrespective of audience duplication. GRPs are stated in terms of percentage: each GRP represents 1% of the people or households tuned to a TV program (as compared to the total number of TV sets in the market being studied). GRPs are calculated by multiplying the total reach of the media schedule by its frequency. Thus the product of reach and frequency expresses the gross duplicated percentage of audience that would be reached by the advertising vehicle. Comparisons of different media vehicles are done based on their respective cost per gross rating point.

Groupon

H

Hard Refresh

When navigating a website, a hard refresh bypasses the web browse's local cache and forces a new request to the server. This is typically done by holding the "Shift" key while clicking the refresh or reload button in the web browser.

Health Monitoring

Health Monitoring is an ASP.NET feature implemented on Marketing IT managed websites to send emails any time an unhandled exception occurs.

Home Page

The webpage that is displayed by default when a visitor surfs directly to the site using a web browser, e.g. the page shown when surfing to http://www.orkin.com.

A pest and termite control company acquired by Rollins in 2008. Home Team uses PestPac for account management and online self service.

Hostname

A label that is assigned to a device connected to a computer network and that is used to identify the device in various forms of electronic communication such as the World Wide Web or e-mail.

HTTP Referrer

The HTTP referrer, commonly referred to as simply referrer, is an HTTP header field that identifies, from the point of view of a web page or other web resource, the address of the webpage (commonly the URL) of the resource which links to it. By checking the referrer, the new page can see where the request came from. Referrer logging is used to allow websites and web servers to identify where people are visiting them from, for promotional or security purposes.

Household Income (HHI)

The total income for all members of a household. Used for demographic targeting.

Hyperlink

Also known as a link, a hyperlink is a reference to a document that the reader can directly follow. The reference points to another document or to a specific element within a document. The most common example of hypertext today is the World Wide Web: web pages contain hyperlinks to other web pages.

Hypervisor

 A hypervisor, also called virtual machine monitor (VMM), is a computer software/hardware platform virtualization software that allows multiple operating systems to run on a host computer concurrently.

I

Impression

The event of an ad being shown to a consumer, such as a website visitor.

Inbound Forwarded Lead

In ODS, an inbound lead for a service which agents cannot sell over the phone and are subsequently forwarded from Edge to FOCUS. The forwarding action is controlled by the agent and is not configured to automatically forward based on the service.

Inbound Lead

A lead that results from an inbound phone call. See also: Outbound lead.

Inbound Proposed Lead

In ODS, an inbound lead from Edge with a VDN assigned to it where the agent was able to propose and possibly schedule service. This setting is controlled by the agent.

Information Architecture, or IA, has somewhat different meanings in different branches of IS or IT architecture. Within Digital Strategy and Acquisition, Information Architecture typically refers to the structural design and methods of organizing and labeling websites, intranets, and online communities.

Internet Explorer (IE)

A  piece of content or webpage that overtakes the expected content immediatly before or after it is viewed, usually as a form of advertising.

JavaScript

A scripting language used to enable programmatic access to objects within other applications. It is primarily used in the form of client-side JavaScript for the development of dynamic websites. JavaScript is a dialect of the ECMAScript standard and is characterized as a dynamic, weakly typed, prototype-based language with first-class functions. A fast and concise JavaScript library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.

Keyword

The term or phrase typed into a search engine to conduct a search, sometimes referred to as a query.

Key Buying Factor (KBF)

An important benefit and cost attribute that customers evaluate and value when choosing among competing brands.

Key Operating Initiative (KOI)

An initiative with high potential impact on business operations. 

L

Landing Page

A page on a website where one is taken after clicking on a advertisement. Landing pages are designed to expand on the service or product mentioned in an initial advertisement (such as through Google AdWords) in order to convince the consumer to make contact with Orkin. All landing pages use a Marketing Campaign ID to determine the phone number and other information displayed on the page, if a Campaign ID is not specified on the URL (using the adid or cm_mmc query string parameter) a default will be used so that the page does not display without a proper call to action.

Link

See hyperlink.

M

Management Performance Report (MPR)

An internal report run by branches that contains key performance indicators such as leads, sales, starts, staffing levels, and more.

Marin

A PPC bid automation platform that automates common tasks related to PPC campaign management and uses click, lead, and revenue data to create an optimized bidding strategy based on predictive algorithms.

Marin Activity Log

A list of all activities performed in Marin including which user performed what action and when.

Marin Conversion File

A CSV formatted file in which each row contains an aggregate of date, Marin keyword ID, creative ID, keyword match type, number of web sales, number of web starts, total web revenue, number of phone sales, number of phone starts, and total phone revenue.

Marin keyword ID

A value that uniquely identifies a keyword in Marin.

Marketing Campaign

A marketing campaign consists of a marketing campaign ID, a phone number (for offline and/or online), pest control offer, termite control offer, and disclaimers for both offers. Both pest and termite control offers are needed even if the campaign is intended for only one service line; this is intended to help with cross-sales. Marketing campaigns are managed in the Marketing Command Center.

Marketing Campaign ID

The unique identifier for a marketing campaign. Sometimes referred to as a web code, ad ID, or promo code.

Marketing Command Center (MCC)

A web-based application that provides a centralized location for managing marketing data (usually in the Rollins Integration Database).

Marketing Category

In the context of the Marketing Command Center, a Marketing Category is the lowest point in the marketing performance reporting hierarchy. Marketing Categories belong to Marketing Sub Programs and contain Marketing Campaigns.

Marketing Mix

The relative use and impact of the set of tools available to market a product of service. Often described at a high level by the "Four Ps" of Product, Price, Promotion, and Place or the "Four Cs" of Commodity, Cost, Communication, and Channel.

Marketing Mix Analysis (MMA)

A comparison between a Marketing Mix Model (MMM) and actual performance. The outcome of such an analysis is often changes to the model or mix, to bring the two in line.

Marketing Mix Model (MMM)

A statistical analysis on sales and marketing time series data to estimate the impact of various marketing tactics (marketing mix) on sales and then forecast the impact of future sets of tactics. It is often used to optimize advertising mix and promotional tactics with respect to sales revenue or profit.

Marketing Program

In the context of the Marketing Command Center, a Marketing Program is the highest point in the marketing performance reporting hierarchy. Marketing Programs contain Marketing Sub Programs.

Marketing Sub Program

In the context of the Marketing Command Center, a Marketing Sub Program is the middle point in the marketing performance reporting hierarchy. Marketing Sub Programs belong to Marketing Programs and contain Marketing Categories.

Media Content

Generally refers to website content that is an image, audio, or video.

MicroCharts

A set of Excel components from BonaVista Systems that were once used to create our Dashboards.

Microsite

A microsite, also known as a minisite or weblet, is a web design term referring to an individual web page or cluster of pages which are meant to function as an auxiliary supplement to a primary website 

Minimum Viable Product (MVP)

A strategy used for fast, quantitative market testing of a product or product feature. It may also involve carrying out market analysis before hand.

Minisite

Net Promoter Score (NPS)

A score based on the fundamental perspective that every company’s customers can be divided into three categories: Promoters, Passives, and Detractors. The score is calculated by asking on a scale of 0-10: "How likely is it that you would recommend [your company] to a friend or colleague?" Results are categorized into Promoters (9-10), Passives (7-8), and Detractors (0-6). The NPS is then the percentage of promoters minus the percentage of detractors. More information at http://www.netpromoter.com/why-net-promoter/know/.

A highly normalized database, primarily used as the raw data-source for reporting, that pulls from multiple data sources, such as Edge and FOCUS, and creates meaningful relationships between otherwise disparate systems. Data from ODS is aggregated into data marts, which reports are run against.

Outbound Lead

A lead that resulted from an outbound phone call. Typically, these phone calls are initiated by someone filling out a form online. These leads are therefore sometimes called web leads or landing page leads.

Outbound-Web LP Proposed

In ODS, an outbound lead from Edge with a web code assigned to it where the agent was able to propose and possibly schedule service. This setting is controlled by the agent.

Over the Page (OTP) Ad

Sometimes referred to as "takeovers", OTP ads are online advertisements that overlay a web page or some portion of it.

P

Pay Per Click (PPC)

An Internet advertising model used on websites, in which advertisers pay their host only when their ad is clicked. With search engines, advertisers typically bid on keyword phrases relevant to their target market.

PC Offer

An offer for a discount on pest control.

PDF File

A PDF file is a "Portable Document Format" file that is proprietary to Adobe. PDF files can be read with Adobe Reader.

Per Visit Value (PVV)

Sometimes referred to as Value Per Visit (VPV), the per visit value of a website is the revenue generated divided by the number of visits for a given period of time. For example, if there were 1,000 visits to a website in one week and $2,000 in revenue were generated from the website in that same week then PVV = $2,000/1,000 = $2.

Phone Lead

A phone lead, as opposed to a web lead, is a lead that results from an in-bound call to Orkin (or other subsidiary).

Point of View (POV)

Generally refers to a document (formal or informal) describing the perceived risks and benefits of a prospective change to a process or system as it relates to the person or group writing the document.

PRIZM

A consumer segmentation system from Nielsen Claritas. PRIZM combines demographic, consumer behavior, and geographic data to help marketers identify, understand and target their customers and prospects. PRIZM defines every U.S. household in terms of 66 demographically and behaviorally distinct types, or "segments," to help marketers discern those consumers’ likes, dislikes, lifestyles and purchase behaviors.

Product Backlog Item (PBI)

A description of functionality to be delivered as part of a product, usually tied to the specific value it provides to the client and/or end user. PBIs often come in the form of user stories. PBIs are different from tasks in that PBIs describe "what" must be delivered, whereas tasks describe "how" a PBI is delivered.

PSD File

A document created by Adobe Photoshop, the most commonly used professional image editing program. PSD files can include image layers, adjustment layers, layer masks, annotation notes, file information, keywords, and other Photoshop-specific elements.

PSD2HTML

A service which "slices" PSD files into HTML, CSS, JavaScript, and image assets.

Purchase Order Numer (P.O. Number)

A reference number to a purchase order. P.O. numbers are used in the field to process non-received payments as proof that the payment has been sent (typically, this is limited to commercial accounts).

Q

Query string

In the World Wide Web, a part of a URL that contains data to be passed to web applications and websites (the part after the question mark).

R

Reach

Reach is a measurement of the size of the audience to whom you will communicate (or to whom you have communicated) with your advertising schedule. It is the total of the readers, listeners, viewers, or surfers of every medium in which you will run (or have run) an ad or commercial.

A family of web feed formats used to publish frequently updated works—such as blog entries, news headlines, audio, and video—in a standardized format.

Smart Phone

A mobile phone offering advanced capabilities beyond a typical mobile phone, often with PC-like functionality. There is no industry standard definition of a smartphone. For some, a smartphone is a phone that runs complete operating system software providing a standardized interface and platform for application developers. For others, a smartphone is simply a phone with advanced features like e-mail and Internet capabilities, and/or a built-in full keyboard or external USB keyboard and VGA connector. In other words, it is a miniature computer that has phone capability.

SQL Injection Attack

A hacking technique that exploits a security vulnerability occurring in the database layer of an application or website. Top Level Domain (TLD)

A top-level domain (TLD) is one of the domains at the highest level in the hierarchical Domain Name System of the Internet.  For all domains in lower levels, it is the last part of the domain name, that is, the last label of a fully qualified domain name. For example, in the domain name www.example.com, the top-level domain is .com.

Type-In Traffic

User Story

In product management, a user story is one or more sentences in the everyday language of the end user or user of a system that captures what a user does or needs to do as part of his or her job function. User stories are used with agile product development methodologies as the basis for defining the functions a business system must provide, and to facilitate requirements management. It captures the "who", "what" and "why" of a requirement in a simple, concise way, often limited in detail by what can be hand-written on a small paper notecard. utm_campaign (query string parameter)

A query string parameter that specifies how Google Analytics should track visits to the page. The utm_campaign parameter must be set to the campaign ID for the page to render the correct phone number, offer, and other details shown to the user and should be in ALL CAPS with no spaces. 

utm_content (query string parameter)

A query string parameter that specifies how Google Analytics should track visits to the page. The utm_content parameter is required by the BI URL Tagging Process, and is used to indicate the "friendly" name of a marketing campaign (as opposed to the campaign ID which is specified in the utm_campaign parameter). 

utm_medium (query string parameter)

A query string parameter that specifies how Google Analytics should track visits to the page. The utm_medium parameter is required by the BI URL Tagging Process and is used to indicate the program name which the marketing campaign roles up into 

utm_source (query string parameter)

A query string parameter that specifies how Google Analytics should track visits to the page. The utm_source parameter is required by the BI URL Tagging Process and is used to indicate the partner through whom the marketing campaign is being executed or managed (e.g. Google, Imagitas, etc.). 

utm_term (query string parameter)

A query string parameter that specifies how Google Analytics should track visits to the page. The utm_term parameter is optional, and is used when it is necessary indicate the keywords purchased or searched-for in an SEM campaign.

Web Bowser

A web browser, or simply a "browser" for short, is a software application for retrieving, presenting, and traversing information resources on the World Wide Web. An information resource is identified by a Uniform Resource Identifier (URI) and may be a web page, image, video, or other piece of content. Hyperlinks present in resources enable users to easily navigate their browsers to related resources.

Although browsers are primarily intended to access the World Wide Web, they can also be used to access information provided by Web servers in private networks or files in file systems.
