Using the Marketing Command Center

Use the Marketing Command Center to view lead data, manage landing pages and vanities, create reports, and download lead data.

This topic also explains the processes driving the collection of web and phone lead data.

For assistance with Marketing Command Center features, or to request a custom report, contact Mark Williams.

Viewing Lead Data

The Marketing Command Center enables you to view data for Orkin, Orkin Canada, Mobile, and Specialty Brands.

The Home page displays a high level view of current leads in a variety of data configurations.

Click Orkin to view web leads and phone leads for the Orkin brand.

Click Orkin Canada to view web leads for the Orkin Canada brand.

Click Speciality Brands and select a brand from the drop-down menu to view web leads for that brand.

Leads Tracking

Rollins Marketing uses a web-and-phone lead tracking solution that provides data collection and business intelligence functions, used to optimize SEM programs to the keyword level.

See Phone and Web Leads Tracking for information on the processes behind leads tracking.

Managing Leads

To manage leads, click the Marketing button and select Leads from the drop-down menu. A robust report of leads is displayed.

Use the Grouping menu to group the data by campaign, service, city, or other criteria.

Isolate data by date using the date range selectors.

Use the Campaign Search function to find a specific campaign.

Use the filter functions in the report header to find leads by Lead ID, service, state, or other criteria.

After filtering and grouping the data, click Download in Excel Format to capture the data in a spreadsheet.


.

Managing Offers

To manage offers, click the Marketing button and select Offers from the drop-down menu. A robust report of offers is displayed.

Use the filter functions in the report header to find offers by name, last update date, or other criteria.

Use the buttons in the lower left of the grid to add new records, edit an existing record, view record details, or refresh the grid after changes.

Managing Programs

To manage programs, click the Marketing button and select Programs from the drop-down menu. A robust report of programs is displayed.

Click the Program column header to view the data is descending or ascending order.

Use the arrows on each row to drill futher down into the program data.

Use the buttons in the lower left of the grid to add new records, edit an existing record, view record details, or refresh the grid after changes.

 

Managing Campaigns

To manage campaigns, click the Marketing button and select Campaigns from the drop-down menu. A robust report of campaigns is displayed.

Use the Grouping menu to group the data by program, category, or other criteria.

Use the filter functions in the report header to find campaigns by campaign ID, date, VDN, or other criteria.

Use the buttons in the lower left of the grid to add new records, edit an existing record, view record details, or refresh the grid after changes.

 

After filtering and grouping the data, click Download in Excel Format to capture the data in a spreadsheet.

 

Managing Communications

To manage communications, click the Communications button. A selection of drop-down communications is displayed.

Isolate data by date using the date range selectors.

Click a communication to view its data; for example, AskTheOrkinMan.

Click Download in Excel Format to capture the data in a spreadsheet.

Managing Branches

To manage branches, click the Branches button. A robust report of branches is displayed.

Use the Grouping menu to group the data by region, city, or other criteria.

Use the filter functions in the report header to find leads by branch name, state, or other criteria.

After filtering and grouping the data, click Download in Excel Format to capture the data in a spreadsheet.

Click Export ORKIN Google Feed to download the feed as a TXT file.

Use the buttons in the lower left of the grid to view record details or refresh the grid.

 

Managing Reports

Use the Reports function to create and save custom reports that are automatically emailed to you.

Use the buttons in the lower left of the grid to add a report, edit a report, or delete a report.

To create a report, follow these steps:

  1. Click the Reporting button. The Reports List is displayed.
  2. Click the + button in the lower left of the grid. The Report Manager is displayed.
  3. Name the report.
  4. Select a report type; for example, Week to Date.
  5. Enter an email address for delivery. Use a semi-colon to separate multiple email addresses.
  6. Select a delivery day and time.
  7. Click in the Filter Area to add search criteria; for example, Get Records Where Bid Keyword starts with pest.
  8. Click Save Report.

Creating and Updating Landing Pages

To create a landing page, follow these steps:

  1. Select a brand.
  2. Select a brand and a service category.
  3. Enter a redirect URL.
  4. In the Select a Theme drop-down menu, select from the list of official branded templates.
  5. Enter the page title and the vanity name.
  6. Click Copy to duplicate this landing page.
  7. Click Finish to create a new landing page.

To update an existing landing page, follow these steps:

  1. Click the Landing Page button and select Update Landing Page from the drop-down menu.
  2. Select a brand and a service category from the drop-down menus.
  3. Use the Select a Web Page drop-down menu to select an existing landing page for this brand.
  4. Select the zone you want to edit from the Select a Zone drop-down menu.
  5. Edit the content, then click Finish.

Managing Vanities

To manage vanity URLS, click the Manage Vanity button. A list of vanities is displayed.

Use the Search Vanity function to find a specific vanity.

Use the filter functions in the table header to find vanities by vanity URL, modified date, or other criteria.

Click Download in Excel Format to capture the vanity data in a spreadsheet.

Use the buttons in the lower left of the grid to add new records, edit an existing record, view record details, delete records, search for records, or refresh the grid after changes.
